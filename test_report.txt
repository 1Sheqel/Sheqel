2. Формат скачивания видео исправлен на 'bestvideo+bestaudio/best'
   + добавлен _YTLogger для перехвата ошибок постпроцессора.

3. Добавить проверку длины текста перед отправкой в ElevenLabs
   (лимит 5000 символов на запрос).

4. version.json на GitHub — обязательно добавить sha256 перед
   каждым релизом для безопасного автообновления.

5. .venv в проекте привязан к старому пути — пересоздать виртуальное
   окружение или использовать системный python3.11.
