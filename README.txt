SheqelMotion Studio v1.0.1
==========================


БЫСТРЫЙ СТАРТ (Mac)
-------------------
1. Убедись что установлен Python 3.10 или новее
   Скачать: https://www.python.org/downloads/

2. Дважды кликни на start.command
   → сам установит зависимости и запустит приложение

При первом запуске macOS может спросить разрешение:
   Системные настройки → Конфиденциальность и безопасность
   → "Всё равно открыть"


НАСТРОЙКА КЛЮЧЕЙ (один раз)
----------------------------
1. Кнопка "API key"  → введи ключ ElevenLabs (elevenlabs.io)
2. Кнопка "Sync key" → введи ключ Sync.so (sync.so)
   Ключи сохраняются автоматически — вводить каждый раз не нужно.


РЕЖИМЫ РАБОТЫ
-------------
Для каждой задачи выбери режим:

  [ 1 видео ]
    Один файл full_voice.mp3 → одно видео → lipsync целиком.
    Можно указать обрезку аудио (от/до в секундах).

  [ Начало + Конец ]
    Один full_voice.mp3 с разделителем ------ внутри текста.
    Приложение автоматически разрежет голос на 2 части:
      — до первого ------ → видео "начало"
      — после последнего ------ → видео "конец"


ПОТОК РАБОТЫ
------------
1. Кнопка "+ Новая задача" — добавить задачу в очередь

2. Для каждой задачи:
   а) "Выбрать full_voice.mp3" — готовая озвучка (mp3 или wav)
      Если ещё нет — кнопка "Сгенерировать голос" создаст её через ElevenLabs
   б) "Текст проверки"     — тот же текст что был озвучен
   в) "Выбрать видео"      — видео для lipsync

3. Кнопка "НАЧАТЬ ОЧЕРЕДЬ" — запускает все задачи

Результат: папка на Рабочем столе → Lipsync_Queue_Output → batch_...


ГЕНЕРАЦИЯ ГОЛОСА (внутри приложения)
--------------------------------------
Кнопка "Сгенерировать голос":
  → выбери голос из списка "📋 Выбрать из списка моих голосов"
  → напиши текст (с ------ там где нужна пауза)
  → нажми "Сгенерировать mp3"
  → файл сохранится на Рабочий стол в папку ElevenLabs_Generated_Voices

Кнопка "🎙 Клонировать голос":
  → загрузи 1-3 минуты чистой речи (mp3, wav, m4a, flac)
  → задай имя голоса
  → голос появится в твоём аккаунте ElevenLabs


РАЗДЕЛИТЕЛЬ ------
------------------
В тексте используй ------ (6 или более дефисов) как паузу/разделитель.

  Пример для режима "1 видео":
    Привет, это начало моего текста.
    ------
    А это продолжение после паузы.

  Пример для режима "Начало + Конец":
    Текст для видео начала.
    ------
    Текст для видео конца.

Приложение заменяет ------ на паузу перед отправкой в ElevenLabs.


ОБНОВЛЕНИЯ
----------
Приложение само проверяет обновления при запуске.
При наличии новой версии — появится диалог с кнопкой "Скачать обновление".
Каждое обновление проверяется по SHA256 — защита от подмены файла.


ТРЕБОВАНИЯ
----------
  Python 3.10 или новее
  ffmpeg / ffprobe — уже в папке приложения (Apple Silicon M1–M4)

  Если у тебя Intel Mac:
    Удали ffmpeg и ffprobe из папки, затем:
    brew install ffmpeg

  Зависимости Python (ставятся автоматически через start.command):
    customtkinter==5.2.2
    requests==2.34.2
    pillow==12.2.0


КАЧЕСТВО ВИДЕО
--------------
  Кодек:    H.264 (libx264), CRF 13, preset slow
  Резкость: unsharp 3×3, 0.6 — шарп после lipsync
  Аудио:    AAC 256 kbps, 44 100 Гц
  Формат:   yuv420p, faststart (работает везде)


ЧАСТЫЕ ОШИБКИ
-------------
  "Не найден ffmpeg/ffprobe"
  → Убедись что ffmpeg и ffprobe лежат рядом с app.py
    Или установи: brew install ffmpeg

  "Sync failed" / 401
  → Проверь Sync API key (кнопка "Sync key" в приложении)

  "ElevenLabs TTS error"
  → Проверь ElevenLabs API key (кнопка "API key" в приложении)
  → Убедись что на аккаунте есть лимиты

  "Не нашёл пауз в full_voice.mp3"
  → В тексте должен быть разделитель ------
  → Файл full_voice.mp3 должен быть сгенерирован из этого же текста

  Приложение не открывается после скрытия (Cmd+H)
  → Кликни на иконку приложения в Dock — окно восстановится
